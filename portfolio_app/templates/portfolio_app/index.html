PROYECTO IRIS

Modelo Clasificador de Flores Iris

El proyecto clasifica flores Iris utilizando aprendizaje automático para predecir la especie según sus características. Con el conjunto de datos Iris como base, proporciona una solución precisa para identificar especies a partir de mediciones específicas.

PROYECTO PRECIO DE CASAS

Modelo para estimar el precio de casas

El proyecto busca crear un modelo predictivo para estimar precios de casas usando técnicas de aprendizaje automático. Se analizarán datos con atributos como tamaño, calidad general, baños y antigüedad.
